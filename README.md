# 🌌 Project Showcase Frontend — 3D Interactive Portfolio

**Dimas Rahmanda Al Farizi** | [@dimasrahmandaalfarizi](https://github.com/dimasrahmandaalfarizi) | UPN Veteran Jawa Timur — Informatika 2024

A 3D interactive portfolio built with **Next.js**, **React Three Fiber**, **GSAP**, and **Zustand**, dynamically powered by the [Project Showcase API](http://localhost/project-showcase-api).

## ✨ Phase 2 Features

- **Dynamic Projects** — Carousel loaded live from `GET /api/projects`
- **AI Chatbot Overlay** — Glassmorphism assistant panel (✨ bottom-right)
- **Difficulty Toggle** — AI explains a project at beginner / intermediate / expert level
- **Analytics Tracking** — Hover & click events sent to `POST /api/analytics/interaction`

## 🛠️ Tech Stack

- Next.js · React Three Fiber · Three.js · GSAP · Zustand · Tailwind CSS

## 🚀 Getting Started

```bash
npm install
npm run dev
```
